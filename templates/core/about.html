How We Started

Website has easy to use interface so that user can navigate the website easily and shop whatever they want. Website provides service categories such as bachelor party, birthday party, children party, fetus detection party, graduation party, farewell party, wedding anniversary party and other parties.

According to the scenario user can select the type of party and then accessories and supplies will be shown to user related to this party which user can select from according to the need and then simply place the order in few clicks. If in case, there is something which user wants to purchase other than the supplies not present in category service, user can go to main accessories and supplies listing and select from there and can place the order.
